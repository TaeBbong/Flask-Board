# Simple-Flask-Board

## Deploy

```bash
$ git clone github.com/TaeBbong/Flask-Board.git
$ git checkout vuln(or master)
$ python3 -m venv venv
$ pip install requirements.txt
$ flask run
```

## Specs

- Simple User / Post CRUD
- Postgres(Backup) / sqlite3
- Blueprint
- bcrypt
- CORS allowed
